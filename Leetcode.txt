Longest Commong Prefix for multiple words in an array:



public class LongestCommonPrefixSimple {

    public static String longestCommonPrefix(String[] strs) {

        if (strs == null || strs.length == 0) {
            return "";
        }

        String result = "";

        // Loop through characters of first word
        for (int i = 0; i < strs[0].length(); i++) {

            char currentChar = strs[0].charAt(i);

            // Compare with same position in other words
            for (int j = 1; j < strs.length; j++) {

                // If index out of range OR characters don't match
                if (i >= strs[j].length() || strs[j].charAt(i) != currentChar) {
                    return result;
                }
            }

            // Character matched in all words
            result += currentChar;
        }

        return result;
    }

    public static void main(String[] args) {
        String[] words = {"flower", "flow", "flight"};
        System.out.println(longestCommonPrefix(words));
    }
}


Converting Roman in String format to Integer Actual Numbers:


public class RomanToInteger {

    public static int romanToInt(String s) {

        int total = 0;

        for (int i = 0; i < s.length(); i++) {

            int current = value(s.charAt(i));

            // Look ahead to next character
            if (i + 1 < s.length() && current < value(s.charAt(i + 1))) {
                total -= current;   // subtract
            } else {
                total += current;   // add
            }
        }
        return total;
    }

    private static int value(char c) {
        if (c == 'I') return 1;
        if (c == 'V') return 5;
        if (c == 'X') return 10;
        if (c == 'L') return 50;
        if (c == 'C') return 100;
        if (c == 'D') return 500;
        return 1000; // M
    }

    public static void main(String[] args) {
        System.out.println(romanToInt("MCMIV"));
    }
}


Find two numbers in a given array which are added to provide the given sum.

class Solution {
    public int[] twoSum(int[] nums, int target) {

        int[] res = new int[2];

        for (int i = 0; i < nums.length; i++) {

            for (int j = i + 1; j < nums.length; j++) {

                if (nums[i] + nums[j] == target) {
                    res[0] = i;
                    res[1] = j;
                    return res; // stop as soon as found
                }
            }
        }
        return res;
    }
}


Checking Number Palindrome:

class Solution {
    public boolean isPalindrome(int x) {

        int reverse = 0;
        int lastDigit = 0;
        int num = x;

        while(num > 0){
            lastDigit = num % 10;

            reverse = reverse * 10 + lastDigit;

            num /= 10;

        }

        if(x == reverse){
            return true;
        }
        else{
            return false;
        }
        
    }
}




Valid parenthesis check,here stack is used since closing brackets are in reverse and LIFO is the perfect operation for
that.


class Solution {
    public boolean isValid(String s) {
        Stack<Character> stack = new Stack<>();

        for (int i = 0; i < s.length(); i++) {
            char c = s.charAt(i);

            // If opening bracket, push to stack
            if (c == '(' || c == '{' || c == '[') {
                stack.push(c);
            } else {
                // If stack is empty â†’ no matching opening bracket
                if (stack.isEmpty()) {
                    return false;
                }

                char top = stack.pop();

                // Check if it matches the current closing bracket
                if ((c == ')' && top != '(') ||
                    (c == '}' && top != '{') ||
                    (c == ']' && top != '[')) {
                    return false;
                }
            }
        }

        // Stack should be empty at the end
        if (stack.isEmpty()) {
           return true;
        } else {
           return false;
        }

}
